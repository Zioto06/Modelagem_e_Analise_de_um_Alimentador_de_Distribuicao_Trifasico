# Modelagem_e_Analise_de_um_Alimentador_de_Distribuicao_Trifasico
Simulação de fluxo de potência em alimentador trifásico radial de 11,4 kV usando Backward-Forward Sweep. Inclui scripts em Python e MATLAB para validar tensões, correntes, potências e perdas nos modelos de carga Z, I e P. Trabalho da disciplina de Distribuição de Energia Elétrica — IFES Guarapari.
